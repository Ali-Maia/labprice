# Resumo Geral dos Casos de Teste — LabPrice

Este documento apresenta uma visão executiva dos testes do projeto LabPrice, cobrindo a API e o front-end.

Os casos detalhados permanecem organizados em:

- [docs/casos de testes/API](docs/casos%20de%20testes/API)
- [docs/casos de testes/e2e](docs/casos%20de%20testes/e2e)

## Cobertura resumida

- **API:** autenticação, precificação, persistência de catálogo e regras de negócio.
- **Front-end:** login, orçamento, salvamento, visualização, atualização e exclusão de produtos.
